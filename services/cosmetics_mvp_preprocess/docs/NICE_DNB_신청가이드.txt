NICE D&B Open API 테스트베드 신청 가이드
========================================

신청 URL: https://openapi.nicednb.com/index.jsp

Step 1. 접속 및 회원가입
  1. https://openapi.nicednb.com/index.jsp 접속
  2. 상단 "회원가입" 클릭
  3. 기업/개인 정보 입력
     - 사업자등록번호 (법인) 또는 주민등록번호 (개인)
     - 담당자명, 연락처, 이메일
  4. 가입 완료

Step 2. 테스트베드 신청
  1. 로그인 후 "마이페이지" → "테스트베드 신청"
  2. 신청 항목:
     - 서비스 선택: "해외기업정보 API"
     - 사용 목적: "수출 바이어 발굴 및 검증 자동화 시스템 구축"
     - 예상 사용량: "월 100건 (PoC 목적)"
  3. 약관 동의 후 신청

Step 3. 승인 및 Key 발급
  - 처리 기간: 당일~1영업일 (자동승인 대부분)
  - 승인 후 마이페이지에서 확인:
    - API Key (Authorization Bearer Token)
    - API Secret (필요시)
    - 사용량 현황

Step 4. API 호출 테스트
  - Base URL: https://openapi.nicednb.com
  - Header: Authorization: Bearer {API_KEY}
  - 무료 제한: 일 100건, 월 100건

주의사항:
  - 테스트베드는 평가 목적으로만 사용 가능
  - 상업적 운영 시 유료 전환 필요
  - 정부지원(수출바우처/데이터바우처) 병행 신청 권장

정부지원 신청 병행 (추천):
  - 수출바우처: https://www.smpp.go.kr
    - 대상: 연매출 150억원 이하 수출기업
    - 지원내용: 바이어 발굴, 마케팅, 상담 등
  - 데이터바우처: https://datavoucher.ktc.re.kr
    - 대상: 중소기업
    - 지원내용: 데이터 구매비용 75% 지원 (최대 1,500만원)
