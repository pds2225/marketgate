## 2. API 엔드포인트 및 호출 정보
================================================================================

### 한국무역보험공사_바이어 검색 (반영완료)
- Endpoint: `https://apis.data.go.kr/B552696/buyer/getBuyerList`
- Method: GET
- 필수 파라미터: serviceKey, pageNo, numOfRows, **country** (국가코드)
- 선택 파라미터: itemName, buyerName, hsCode, industry
- 응답: JSON/XML

### KOTRA 수출유망추천정보 (반영완료)
- Endpoint: 공공데이터포털 API (정확한 엔드포인트는 명세서 확인 필요)
- 파일 데이터로도 제공: `kotra_export_recommend_all.csv`

### KOTRA 국가정보/해외시장뉴스/상품DB/무역사기사례/기업성공사례 (미반영)
- Endpoint: 공공데이터포털 KOTRA API (활용신청 후 확인 가능)
- 서비스 코드: B490001 계열 (정확한 코드는 명세서 확인 필요)
- 활용신청: https://www.data.go.kr 에서 각 데이터셋별 신청

### NIPA 글로벌ICT포털 해외바이어정보 (미반영)
- Endpoint: `https://apis.data.go.kr/` + 별도 서비스코드
- 글로벌ICT포털: https://www.globalict.kr/intro/openApi.do
- 활용신청 후 승인 필요 (자동승인 또는 승인심의)

## 3. tradeKorea/buyKOREA RFQ 데이터 현황
================================================================================

### 결론: data_catalog.csv에는 없으나, 공공데이터포털에 존재

**대한무역투자진흥공사_인콰이어리 정보 (data.go.kr/data/15155499)**
- 이 데이터가 바로 **buyKOREA 플랫폼의 바이어 인콰이어리 (RFQ)**
- 바이어가 직접 "이 제품을 사겠다"고 올린 수요 신호
- data_catalog.csv에는 누락됨 → 추가 필요

**대한무역투자진흥공사_SNS 마케팅 수집 바이어 정보 (data.go.kr/data/15155497)**
- 메타, 링크드인 등 SNS에서 수집한 관심 바이어 데이터
- 간접 수요 신호 포함

**중소벤처기업진흥공단_해외바이어 구매오퍼 정보 (data.go.kr/data/15002299)**
- 이미 반영완료 (SBC 구매오퍼)
- buyKOREA 외 별도 채널의 구매오퍼
