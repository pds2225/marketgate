# MarketGate UI 디자인 제안 — 흰색·블루 테마

> **요청:** 코드 수정 NO / 디자인 제안만 / 검정화면 X / 흰색·블루 계열 / 간단명료·직관적
> **기준:** ChatModePage.jsx 실제 구현 + v4 설계
> **작성일:** 2026-05-03

---

## 1. 디자인 철학

| 원칙 | 적용 |
|------|------|
| **흰 배경 기반** | `#FFFFFF` 메인 배경 — 깨끗하고 전문적인 느낌 |
| **블루 포인트** | `#2563EB` (Tailwind blue-600) — 신뢰·글로벌·AI |
| **여백 충분** | 카드 간 24px, 섹션 간 48px — 숨 쉬는 UI |
| **직관적 흐름** | 위→아래 단일 플로우, 왼쪽→오른쪽 액션 |
| **상태 명시** | `available` / `soon` / `verified` — 색상+텍스트로 즉시 파악 |

## 8. 스타일 요약 (Tailwind 클래스)

| 요소 | 클래스 |
|------|--------|
| **페이지 배경** | `bg-white` |
| **카드 배경** | `bg-white` / 준비중: `bg-gray-50` |
| **카드 테두리** | `border-2 border-blue-600` / 준비중: `border border-gray-200` |
| **카드 둥근모서리** | `rounded-2xl` (16px) |
| **카드 그림자** | `shadow-sm hover:shadow-md` |
| **주 버튼** | `bg-blue-600 text-white px-5 py-2.5 rounded-lg font-semibold` |
| **보조 버튼** | `bg-gray-100 text-gray-700 px-4 py-2 rounded-lg` |
| **AI 버블** | `bg-gray-100 text-gray-900 rounded-2xl rounded-tl-sm` |
| **사용자 버블** | `bg-blue-600 text-white rounded-2xl rounded-tr-sm` |
| **입력창** | `bg-white border border-gray-200 focus:border-blue-600 rounded-xl` |
| **뱃지(사용가능)** | `bg-blue-100 text-blue-700 text-xs font-semibold px-3 py-1 rounded-full` |
| **뱃지(준비중)** | `bg-gray-100 text-gray-500 text-xs font-medium px-3 py-1 rounded-full` |
| **텍스트(메인)** | `text-gray-900 text-base` |
| **텍스트(보조)** | `text-gray-400 text-sm` |
